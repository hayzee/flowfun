# flowfun

Clojurescript starter project with shadow-cljs.

## Usage

1. Install

$ npm install

2. Start the server

$ npm run watch

3. Open browser 

http://localhost:8888/

4. Start clojurescript repl (IntelliJ/Cursive)

First configure and start a new nRepl that connects to the standard port file.

> (shadow/repl :app)

You are now live coding!
